Soundtrak

Project

Soundtrak was a group project in an mock agency project setting. Working form the brief we utalized the MovieDb to alllow the user to filter movies by genre. Once the movie is selected we use the Spotify API to find the soundtrack to this movie. If there aren't any availble another call is made to find an appropriate playlist that spotify users have created.

Technology

React / Javascript / Stotify and MovieDB APIs / Gulp / Scss /Html
