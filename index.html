About

Product designer turned web-developer based out of Toronto. Challenging myself everyday to master new tools and continue to learn in an industry that never stops growing.

Fuelled by caffeine and an innate curiosity and love for all things thoughtfully designed, I find myself focused on usability and clean design. Passionate about creating playful and engaging user experiences.

Vaporwave Album

Project - View Live

Vaporwave is an internet born music genre and art movement that emerged in the early 2010s. It gained a following through the tumblr music scene, with reddit following suit. As a tribute to this “A E S T H E T I C”, Vapor-Art.com allows users to play with images and text to create a piece of A R T.

Soundtrak

Project - View Live

Soundtrak is a group project which required the 3 of us to collaborate through Github. Working form the brief, we utilized the MovieDb to allow users to filter movies by genre. When a movie is selected and the user has been authenticated by Spotify, we use the Spotify API to return original soundtracks or playlist for this movie.

Technology - View Github

React / Javascript / APIs / Gulp / Github / Sass / HTML
